# DRI-FEDS-FIRE

Configuration and execution scripts for running the Fire Event Data Suite (FEDS) on NCAR Derecho.

## Purpose

This repository documents:
- Environment setup on Derecho
- Python and shell scripts used to configure and run FEDS
- Workflow development during February 11–13, 2026
- Reproducibility steps for DRI Research Immersion students

## System

- HPC: NCAR Derecho
- Python: derecho_py kernel
- Data: VIIRS fire detection products
- Project: DRI Wildfire Modeling Initiative

## Structure

- scripts/ : Shell and Python scripts
- docs/ : Workflow documentation
- env/ : Environment configuration
- logs/ : Execution logs (not tracked if large)
